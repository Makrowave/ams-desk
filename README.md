# Ams Frontend

This is a frontend app for bike management system, written in React with Next.js
It makes a complete project alongside [Ams Tel](https://github.com/Makrowave/ams-tel) and [Ams Backend](https://github.com/Makrowave/ams-desk-cs-backend).

Licensed under [Elv2](https://www.elastic.co/licensing/elastic-license).

### At the current state one can:
- Customize application for business' needs
- Add models and manage bikes - their state, placement, quantity.
- Manage bike repair shop - create new repairs, track their state
    add parts and services, print docs and auto calculate costs.

### What's to come:
- In house messaging system
- Status dashboards
- Sale analysis graphs (while it is not a feature yet - data is collected)
- Delivery assistance - integrated with mobile app, quickly add bikes to system
